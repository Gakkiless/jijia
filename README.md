# 🏔 松赞 · 产品计价工具

松赞旅行产品内部计价工具，基于产品计价规则自动计算行程报价。

## 功能

- **行程编码解析** — 输入简称编码（如 `波波百巴巴`），自动解析酒店和天数
- **酒店选择** — 从松赞酒店码表（23个）下拉选择，含区域标签
- **三种产品形态** — 自由行 / 私享管家 / 主题团
- **车司管计费** — 按接送机组合自动计算计费天数
- **儿童折扣** — 按年龄自动计算餐饮折扣（<6岁3折/6-11岁7折）
- **四季成本率** — 节日/旺季/平季/淡季一键切换对比
- **预售折扣** — 按环线 × 长短线自动匹配折扣规则

## 使用

直接打开 `index.html` 即可使用，无需安装依赖。

## 计价公式

```
打包价 = 元素总成本 / 成本率
```

| 季节 | 成本率 | 利润率 |
|------|--------|--------|
| 节日 | 0.80 | 20% |
| 旺季 | 0.825 | 17.5% |
| 平季 | 0.85 | 15% |
| 淡季 | 0.90 | 10% |
